Template Helper Functions

Template Helpers are powerful utility functions available in all HTML templates in Octopus. They provide mathematical operations, data manipulation, formatting, and conditional logic to enhance your template rendering capabilities.

Key Categories:

  • Dictionary & Collection Operations
  • Number Formatting & Mathematics
  • Conditional Logic & Comparisons
  • Random Number Generation
Example Template Usage HTML 
<!-- Template Helper Example -->
{{ "{{" }} $user := dict "name" "John" "age" 30 {{ "}}" }}
{{ "{{" }} $total := addInt .quantity .price {{ "}}" }}
<div class="user-card">
  <h3>{{ "{{" }} $user.name {{ "}}" }}</h3>
  <p>Age: {{ "{{" }} $user.age {{ "}}" }}</p>
  <p>Total: ${{ "{{" }} formatNum $total 2 {{ "}}" }}</p>
  {{ "{{" }} if intGt $user.age 18 {{ "}}" }}
    <span class="adult">Adult</span>
  {{ "{{" }} end {{ "}}" }}
</div>

Dictionary & Collection Functions

dict key1 value1 key2 value2 ...

Creates a dictionary/map from key-value pairs

<!-- Basic dictionary creation -->
{{ "{{" }} $user := dict "name" "John Doe" "age" 30 "email" "john@example.com" {{ "}}" }}
<div class="user-card">
  <h3>{{ "{{" }} $user.name {{ "}}" }}</h3>
  <p>Age: {{ "{{" }} $user.age {{ "}}" }}</p>
  <p>Email: {{ "{{" }} $user.email {{ "}}" }}</p>
</div>

<!-- Nested dictionaries -->
{{ "{{" }} $config := dict
  "theme" "dark"
  "settings" (dict "autoSave" true "notifications" false)
  "colors" (dict "primary" "#007bff" "secondary" "#6c757d")
{{ "}}" }}
<div class="app" data-theme="{{ "{{" }} $config.theme {{ "}}" }}">
  <div style="background-color: {{ "{{" }} $config.colors.primary {{ "}}" }}">
    Auto-save: {{ "{{" }} if $config.settings.autoSave {{ "}}" }}enabled{{ "{{" }} else {{ "}}" }}disabled{{ "{{" }} end {{ "}}" }}
  </div>
</div>

length value

Returns the length of strings, slices, arrays, and maps

<!-- String length -->
{{ "{{" }} $message := "Hello, World!" {{ "}}" }}
<p>Message length: {{ "{{" }} length $message {{ "}}" }} characters</p>

<!-- Array/Slice length -->
{{ "{{" }} $items := .Products {{ "}}" }}
<div class="products-header">
  <h2>Products ({{ "{{" }} length $items {{ "}}" }} items)</h2>
</div>
{{ "{{" }} if gt (length $items) 0 {{ "}}" }}
  <ul>
  {{ "{{" }} range $items {{ "}}" }}
    <li>{{ "{{" }} .Name {{ "}}" }}</li>
  {{ "{{" }} end {{ "}}" }}
  </ul>
{{ "{{" }} else {{ "}}" }}
  <p>No products available</p>
{{ "{{" }} end {{ "}}" }}

<!-- Conditional rendering -->
{{ "{{" }} if gt (length .Comments) 5 {{ "}}" }}
  <div class="comments-summary">
    Showing 5 of {{ "{{" }} length .Comments {{ "}}" }} comments
    <a href="/comments/all">View all</a>
  </div>
{{ "{{" }} end {{ "}}" }}

Number Formatting & Time Functions

formatNum number precision

Formats numeric values to specified decimal places

<!-- Basic price formatting -->
{{ "{{" }} $price := 123.456 {{ "}}" }}
<span class="price">${{ "{{" }} formatNum $price 2 {{ "}}" }}</span>
<!-- Output: $123.46 -->

<!-- Statistics display -->
{{ "{{" }} $stats := dict "average" 87.6789 "total" 1234.5 "percentage" 0.8567 {{ "}}" }}
<div class="stats">
  <div class="stat">
    <label>Average:</label>
    <span>{{ "{{" }} formatNum $stats.average 2 {{ "}}" }}</span>
  </div>
  <div class="stat">
    <label>Total:</label>
    <span>{{ "{{" }} formatNum $stats.total 0 {{ "}}" }}</span>
  </div>
  <div class="stat">
    <label>Percentage:</label>
    <span>{{ "{{" }} formatNum (multiplyFloat $stats.percentage 100) 1 {{ "}}" }}%</span>
  </div>
</div>

timestamp dateString

Converts date strings to Unix timestamps

<!-- Date string to timestamp -->
{{ "{{" }} $dateStr := "2025-06-04T09:41:56.973000" {{ "}}" }}
{{ "{{" }} $ts := timestamp $dateStr {{ "}}" }}
<div class="date-info">
  <p>Date: {{ "{{" }} $dateStr {{ "}}" }}</p>
  <p>Timestamp: {{ "{{" }} $ts {{ "}}" }}</p>
  <p>Formatted: <time datetime="{{ "{{" }} $dateStr {{ "}}" }}">{{ "{{" }} $dateStr {{ "}}" }}</time></p>
</div>

<!-- Conditional based on timestamp -->
{{ "{{" }} $eventDate := timestamp .Event.Date {{ "}}" }}
{{ "{{" }} $now := timestamp "2025-01-01T00:00:00" {{ "}}" }}
{{ "{{" }} if intGt $eventDate $now {{ "}}" }}
  <span class="upcoming">Upcoming Event</span>
{{ "{{" }} else {{ "}}" }}
  <span class="past">Past Event</span>
{{ "{{" }} end {{ "}}" }}

Random Number Generation

Generates random integers (rand: 0-99, randRange: min-max)

<!-- Random styling -->

  <h1>Welcome to Our Site</h1>
</div>

<!-- Random layout columns -->
{{ "{{" }} $columns := randRange 2 5 {{ "}}" }}
<div class="grid grid-cols-{{ "{{" }} $columns {{ "}}" }}">
  {{ "{{" }} range .Items {{ "}}" }}
    <div class="grid-item">{{ "{{" }} .Title {{ "}}" }}</div>
  {{ "{{" }} end {{ "}}" }}
</div>

<!-- Random animations -->
{{ "{{" }} range .Testimonials {{ "}}" }}
  <div class="testimonial"
       style="animation-delay: {{ "{{" }} randRange 0 3 {{ "}}" }}s;

    <p>{{ "{{" }} .Text {{ "}}" }}</p>
    <cite>{{ "{{" }} .Author {{ "}}" }}</cite>
  </div>
{{ "{{" }} end {{ "}}" }}

         style="left: {{ "{{" }} randFloatRange 0 100 {{ "}}" }}%;

    </div>

<!-- Random progress bars -->
{{ "{{" }} range .Skills {{ "}}" }}

Integer Math Functions

addInt, subtractInt, multiplyInt, divideInt a b

Basic integer arithmetic operations

<!-- Shopping cart calculations -->
{{ "{{" }} $quantity := .Quantity {{ "}}" }}
{{ "{{" }} $price := .Price {{ "}}" }}
{{ "{{" }} $total := multiplyInt $quantity $price {{ "}}" }}
{{ "{{" }} $discount := divideInt $total 10 {{ "}}" }} <!-- 10% discount -->
{{ "{{" }} $finalTotal := subtractInt $total $discount {{ "}}" }}

<div class="cart-item">
  <h3>{{ "{{" }} .ProductName {{ "}}" }}</h3>
  <p>Quantity: {{ "{{" }} $quantity {{ "}}" }}</p>
  <p>Price per item: ${{ "{{" }} $price {{ "}}" }}</p>
  <p>Subtotal: ${{ "{{" }} $total {{ "}}" }}</p>
  <p>Discount: ${{ "{{" }} $discount {{ "}}" }}</p>
  <p class="total">Final Total: ${{ "{{" }} $finalTotal {{ "}}" }}</p>
</div>

<!-- Pagination calculations -->
{{ "{{" }} $currentPage := .CurrentPage {{ "}}" }}
{{ "{{" }} $totalPages := .TotalPages {{ "}}" }}
<div class="pagination">
  {{ "{{" }} if intGt $currentPage 1 {{ "}}" }}
    <a href="?page={{ "{{" }} subtractInt $currentPage 1 {{ "}}" }}">Previous</a>
  {{ "{{" }} end {{ "}}" }}
  {{ "{{" }} if intLt $currentPage $totalPages {{ "}}" }}
    <a href="?page={{ "{{" }} addInt $currentPage 1 {{ "}}" }}">Next</a>
  {{ "{{" }} end {{ "}}" }}
</div>

modInt, absInt, maxInt, minInt

Advanced integer operations: modulo, absolute, maximum, minimum

<!-- Grid layout with modulo -->
{{ "{{" }} $itemsPerRow := 3 {{ "}}" }}
{{ "{{" }} range $index, $item := .Items {{ "}}" }}
  {{ "{{" }} $position := modInt $index $itemsPerRow {{ "}}" }}
  <div class="grid-item {{ "{{" }} if intEq $position 0 {{ "}}" }}first{{ "{{" }} end {{ "}}" }}">
    {{ "{{" }} $item.Title {{ "}}" }}
  </div>
  {{ "{{" }} if intEq $position 2 {{ "}}" }}
    <div class="row-break"></div>
  {{ "{{" }} end {{ "}}" }}
{{ "{{" }} end {{ "}}" }}

<!-- Temperature display with absolute values -->
{{ "{{" }} $temp := -15 {{ "}}" }}
<div class="temperature">
  <span>{{ "{{" }} $temp {{ "}}" }}°C</span>
  <span class="abs-temp">{{ "{{" }} absInt $temp {{ "}}" }}° from zero</span>
</div>

<!-- Rating calculations -->
{{ "{{" }} $rating1 := 4 {{ "}}" }}
{{ "{{" }} $rating2 := 5 {{ "}}" }}
{{ "{{" }} $rating3 := 3 {{ "}}" }}
{{ "{{" }} $maxRating := maxInt (maxInt $rating1 $rating2) $rating3 {{ "}}" }}
{{ "{{" }} $minRating := minInt (minInt $rating1 $rating2) $rating3 {{ "}}" }}
<div class="ratings">
  <p>Highest: {{ "{{" }} $maxRating {{ "}}" }} stars</p>
  <p>Lowest: {{ "{{" }} $minRating {{ "}}" }} stars</p>
</div>

Best Practices & Tips

Type Safety

Always use the appropriate function for your data type (intAdd vs floatAdd) to ensure predictable results.

Performance

Store complex calculations in variables to avoid repeated computation within loops.

Formatting

Use formatNum for consistent decimal places in financial and statistical displays.

Conditionals

Combine comparison functions with template conditionals for dynamic styling and content.
